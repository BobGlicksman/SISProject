SIS - Smart Intelligent Sensor

Welcome to the SIS Control center. The Standalone Intelligent Sensor is an open collaboration between Bob Glicksman and Jim Schrempp. It is a platform to make it simple (relatively speaking) to add remote monitoring to any location. Today the SIS runs on a Photon from Particle.io, inexpensive off the shelf wireless sensors, and a bit of Javascript to run a control web page.

It is all available on GitHub.

Custom Configs

These are config pages that have been customized to a particular use case. The only difference over the standard page is that the number of sensors and sensor names may have been changed, and perhaps the sensor order. This customization is all done easily with some constants at the top of the Javascript file.
