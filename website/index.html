SIS - Smart Intelligent Sensor

Welcome to the SIS Control center. The Smart Intelligent Sensor is an open collaboration between Bob Glicksman and Jim Schrempp. It is a platform to make it simple (relatively speaking) to add remote monitoring to any location. Today the SIS runs on a Spark Core from Spark.io and a bit of Javascript to run a control web page.

Two control pages are available:

  • Configure a Small Apartment

  • Configure Saratoga Townhouse

  • Configure Elm Street

  • Raw Debugging
